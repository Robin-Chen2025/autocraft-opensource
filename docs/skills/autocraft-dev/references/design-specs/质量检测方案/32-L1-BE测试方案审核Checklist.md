# L1-BE审核 Checklist — V2

**版本：** v2.0
**日期：** 2026-05-13
**适用文档：** L1-BE测试方案

---

## E系列：阻断项（每项10分）

| ID | 检查内容 | 通过标准 |
|----|---------|---------|
| E01 | 触发规则分析存在 | 明确列出哪些代码命中了哪条规则 |
| E02 | 未命中模块说明 | 明确列出哪些模块不需要L1及原因 |
| E03 | 异常场景清单存在 | 每条命中规则有对应测试场景 |
| E04 | 规则1场景覆盖 | 超时+格式异常+错误码+空值 |
| E05 | 规则2场景覆盖 | 正常流转+非法跳转+回退 |
| E06 | 规则3场景覆盖 | 字段缺失+类型错误+嵌套变化 |
| E07 | Mock范围正确 | 仅mock外部服务，不mock内部逻辑 |

## W系列：警告项（每项2分）

| ID | 检查内容 | 通过标准 |
|----|---------|---------|
| W01 | 触发关键词列表完整 | 列出代码中命中的关键词 |
| W02 | 并发场景覆盖 | 规则2命中时考虑并发冲突 |

**评分**：总分 = 100 - (E缺失 × 10) - (W缺失 × 2)
