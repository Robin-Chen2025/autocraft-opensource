## 使用说明

**评分计算**：
```
总分 = 100 - (阻断项缺失数 × 10) - (警告项缺失数 × 2)
```

**评分等级**：
- ≥90分：通过，定稿
- 70-89分：需修改，修复阻断项后重新审核
- <70分：不通过，重新生成

## 六、快速审核流程

```
Step1: 检查E01-E06（文档结构）
    └─ 缺失任一 → 不通过，重新生成

Step2: 检查E07-E12（L1测试规范）
    └─ L1规范缺失 → 开发计划制定无依据

Step3: 检查E13-E24（场景矩阵+验收标准）
    └─ 统计缺失数量 → 计算扣分

Step4: 检查W01-W10（数据依赖+文档引用）
    └─ 统计缺失数量 → 计算扣分

Step5: 计算总分 → 判断结论

Step6: 填写缺失项清单 → 提供修复建议

Step7: 签字确认
```
