## 使用说明

**评分计算**：
```
总分 = 100 - (阻断项缺失数 × 10) - (内容审核缺失数 × 5) - (警告项缺失数 × 2)
```

**评分等级**：
- ≥90分：通过，定稿
- 70-89分：需修改，修复阻断项+内容审核缺失项后重新审核
- <70分：不通过，重新生成

## 七、快速审核流程

```
Step1: 检查E01-E05（文档结构）
    └─ 缺失任一 → 不通过，重新生成

Step2: 检查E06-E30（内容完整性）
    └─ 统计缺失数量 → 计算扣分

Step3: 检查C01-C30（内容审核-页面流程模拟验证）
    ├─ 对照UI设计文档验证页面覆盖
    ├─ 对照业务流程文档验证页面串联
    ├─ 对照组件规范文档验证组件交互
    ├─ 对照PRD验证功能覆盖
    └─ 统计缺失数量 → 计算扣分

Step4: 检查W01-W15（细节完整性）
    └─ 统计缺失数量 → 计算扣分

Step5: 计算总分 → 判断结论

Step6: 填写缺失项清单 → 提供修复建议

Step7: 签字确认
```

**内容审核对照文档清单**：

| 对照文档 | 提取内容 | 对应检查项 |
|---------|---------|-----------|
| UI设计文档 | P0页面清单、页面跳转、表单字段 | C01-C05、C26-C30 |
| 业务流程设计文档 | 流程节点、分支、异常跳转 | C06-C10 |
| 组件规范文档 | 公共组件、表单验证、按钮状态 | C11-C15 |
| 数据库设计文档/API设计文档 | 输入输出字段匹配 | C16-C20 |
| PRD | P0功能、验收标准、用户交互 | C21-C25 |
